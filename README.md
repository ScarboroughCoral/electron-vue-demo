# electron-vue-demo
> Vue-cli Electron ElementUI
## 安装依赖
```
yarn
```
## 运行与打包
### 热更新开发模式
```
yarn run electron:serve
```

### 生产模式打包
```
yarn run electron:build
```

### Lints and fixes files
```
yarn lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
